# Multi_Cloud
This is a sample Github Repo for Multi-Cloud. For Azure, AWS and GCP
1. Create virtual environment and source
    python3 -m venv ~/.Multi_cloud
    source ~/.flask-ml-azure/bin/activate
2. Run make install

3. Run python hello.py
